# Student-Management-App

The Student Management App is a simple web application built using React and Vite. It allows users to manage student records by adding, viewing, and deleting student information. Each student record includes details such as Name, Age, and Course.

This project demonstrates the fundamentals of React development, including component-based architecture, state management using React Hooks (useState), form handling, props, and dynamic rendering of data. It is designed for beginners who want to learn how to build interactive user interfaces with React.

# Features
Add new student records
Display a list of students
Delete student records
Reusable React components
State management using React Hooks
Responsive and user-friendly interface
Built with React + Vite for fast development

# Technologies Used
React
Vite
JavaScript (ES6+)
CSS3
HTML5

# Learning Outcomes
Through this project, students will learn:
Creating React applications using Vite
Building reusable components
Managing application state with useState
Handling forms and user input
Rendering dynamic lists
Passing data using props
Implementing CRUD basics (Create, Read, Delete)

# Conclusion
This project provides a strong foundation in React development and helps beginners understand how modern frontend applications are built using reusable components and state-driven user interfaces.
